# LEVEL 0

A first-person browser horror experience built with Three.js. Every run creates a new connected Backrooms maze with procedural wallpaper, carpet, ceiling tiles, fluorescent lighting, spatial audio, environmental scares, a stalking entity, and a reachable EXIT.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, select **ENTER LEVEL 0**, and allow pointer lock.

## Controls

- `WASD` or arrow keys: move
- Mouse: look
- `Shift`: run
- `Ctrl` or `C`: crouch
- `E`: open the EXIT
- `Esc`: pause and release the mouse

Touch devices use a left movement pad, right-side swipe look, a run button, and a contextual open button.

## Production build

```bash
npm run build
npm run preview
```

The production output is written to `dist/`.
