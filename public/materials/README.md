# Generated material library

These project-local albedo sources were generated with the built-in OpenAI image-generation workflow, resized to 1024×1024, and converted to JPEG for browser delivery.

- `aged-wallpaper-albedo.jpg` — seamless aged commercial vinyl wallpaper.
- `damp-carpet-albedo.jpg` — seamless worn low-pile commercial carpet.
- `acoustic-tile-albedo.jpg` — seamless mineral-fiber ceiling tile.
- `survivor-fabric-albedo.jpg` — neutral seamless protective-suit fabric used with runtime color tinting.
- `creature-skin-albedo.jpg` — neutral seamless creature-skin detail used as a bump source.

The environment textures are blended with deterministic procedural color and damage in `src/main.js`, so level identity and multiplayer topology remain configuration-driven. The survivor and creature maps affect presentation only.

Generation constraints shared by the prompts: photorealistic PBR-style material capture, orthographic square framing, flat neutral lighting, seamless edges, no focal object, no text, no logos, and no watermark.
